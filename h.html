برنامه‌نویسی (پایتون)

یک راهنمای کامل و تصویری برای یادگیری زبان برنامه‌نویسی پایتون.

درس ۱: سفر هیجان‌انگیز به دنیای پایتون

پایتون یک زبان برنامه‌نویسی قدرتمند، همه‌کاره و آسان برای یادگیری است که در زمینه‌های مختلفی از جمله توسعه وب، تحلیل داده و هوش مصنوعی کاربرد دارد.

درس ۲: شروع ماجراجویی

ماجراجویی برنامه‌نویسی شما با پایتون با نصب ابزارهای لازم شروع می‌شود.

درس ۳: نصب پایتون

برای شروع کار، ابتدا باید پایتون را روی سیستم خود نصب کنید. به وب‌سایت رسمی پایتون بروید و آخرین نسخه پایدار را دانلود کنید. هنگام نصب، حتماً گزینه "Add Python to PATH" را فعال کنید.

درس ۴: نصب IDE

یک محیط توسعه یکپارچه (IDE) مثل VS Code یا PyCharm به شما در نوشتن و مدیریت کد کمک می‌کند.

درس ۵: انجام یکسری تنظیمات مهم

بعد از نصب، چند تنظیمات کوچک در IDE می‌تواند تجربه برنامه‌نویسی شما را بهبود ببخشد.

درس ۶: آشنایی با shell پایتون

شِل پایتون یک محیط تعاملی است که به شما اجازه می‌دهد کد را خط به خط اجرا کنید.

>>> 2 + 2
4
>>> "hello" + " world"
'hello world'

مثال ۲: استفاده از متغیرها در شل

>>> x = 10
>>> print(x * 5)
50

درس ۷: متغیر یا Variable

متغیرها برای ذخیره داده‌ها در حافظه استفاده می‌شوند. شما می‌توانید هر نوع داده‌ای را در یک متغیر ذخیره کنید.

name = "Sina"
age = 22

مثال ۲: تغییر مقدار یک متغیر

x = 10
x = x + 5 # حالا مقدار x برابر 15 است.

درس ۸: انواع داده‌ها در پایتون - datatypes

پایتون دارای انواع داده مختلفی است، از جمله رشته (String)، عدد (Integer و Float)، بولین (Boolean) و ... .

# مثال از انواع داده
text = "Hello" # رشته
number = 100 # عدد صحیح
decimal = 10.5 # عدد اعشاری
is_valid = True # بولین

مثال ۲: چک کردن نوع داده

print(type(text))
print(type(number))

درس ۹: جدول انواع داده‌ها

این جدول به شما کمک می‌کند تا انواع داده‌های اصلی را بهتر بشناسید.

درس ۱۰: اجرای فایل پایتونی در محیط vs_code

با استفاده از ترمینال داخلی VS Code، می‌توانید فایل پایتون خود را به راحتی اجرا کنید. کافیست روی فایل راست کلیک کرده و "Run Python File in Terminal" را انتخاب کنید.

درس ۱۱: اجرای فایل پایتونی در محیط cmd ویندوز

برای اجرای فایل در محیط CMD، ابتدا با دستور `cd` به پوشه‌ای که فایل در آن قرار دارد بروید و سپس دستور `python your_file.py` را اجرا کنید.

cd C:\Users\YourName\Documents
python my_program.py

درس ۱۲: کار با اعداد و محاسبات ریاضی در پایتون

پایتون از تمامی عملگرهای ریاضی مانند جمع، تفریق، ضرب و تقسیم پشتیبانی می‌کند.

result_sum = 10 + 5
result_sub = 10 - 5
result_mul = 10 * 5
result_div = 10 / 5
print(f"جمع: {result_sum}, تفریق: {result_sub}, ضرب: {result_mul}, تقسیم: {result_div}")

مثال ۲: استفاده از توان و باقیمانده

power = 2 ** 3 # 2 به توان 3
remainder = 10 % 3 # باقیمانده تقسیم 10 بر 3
print(f"توان: {power}, باقیمانده: {remainder}")

درس ۱۳: اولویت عملگرها

پایتون از قوانین استاندارد ریاضی برای اولویت عملگرها (PEMDAS) پیروی می‌کند. پرانتز، توان، ضرب/تقسیم، و سپس جمع/تفریق.

result = 10 + 5 * 2 # ابتدا ضرب انجام می‌شود
print(result) # 20

مثال ۲: استفاده از پرانتز برای تغییر اولویت

result = (10 + 5) * 2 # ابتدا جمع انجام می‌شود
print(result) # 30

درس ۱۴: کار با استرینگ‌ها

رشته‌ها (Strings) برای ذخیره متن استفاده می‌شوند. آن‌ها با نقل‌قول دوتایی یا تکی تعریف می‌شوند.

text1 = "Hello Python"
text2 = 'آموزش پایتون'
combined = text1 + " - " + text2
print(combined)

مثال ۲: تکرار رشته

repeated_text = "آها " * 3
print(repeated_text) # آها آها آها

درس ۱۵: مفهوم ایندکسینگ و استرینگ‌ها

هر کاراکتر در یک رشته یک ایندکس (موقعیت) دارد که با صفر شروع می‌شود.

my_string = "Python"
print(my_string[0]) # 'P'
print(my_string[2]) # 't'
print(my_string[-1]) # 'n' (آخرین کاراکتر)

مثال ۲: برش (Slicing) رشته‌ها

slice_text = "Hello World"
print(slice_text[0:5]) # 'Hello'
print(slice_text[6:])  # 'World'

درس ۱۶: متدهای پرکاربرد در string

متدها توابعی هستند که روی اشیاء عمل می‌کنند.

name = "alireza"
print(name.upper()) # 'ALIREZA'
print(name.capitalize()) # 'Alireza'

مثال ۲: جایگزینی و یافتن متن

sentence = "I love Python"
new_sentence = sentence.replace("Python", "Programming")
print(new_sentence) # 'I love Programming'
print(sentence.find("Python")) # 7 (ایندکس شروع)

درس ۱۷: مفهوم کامنت‌گذاری در پایتون

کامنت‌ها خطوطی از کد هستند که توسط مفسر پایتون نادیده گرفته می‌شوند و برای توضیحات استفاده می‌شوند. آن‌ها با `#` شروع می‌شوند.

# این یک کامنت است که هدف کد را توضیح می‌دهد.
x = 10 # این کامنت هم در کنار کد قرار می‌گیرد.
print("Hello")

مثال ۲: استفاده از چندخطی برای کامنت‌ها

"""
این یک کامنت چند خطی است
که برای توضیحات طولانی‌تر استفاده می‌شود.
"""
print("World")

درس ۱۸: داده نوع Boolean

داده‌های بولین فقط دو مقدار `True` و `False` دارند و برای منطق و تصمیم‌گیری استفاده می‌شوند.

is_adult = True
is_cold = False
if is_adult:
    print("شما بزرگسال هستید.")

مثال ۲: مقایسه و خروجی بولین

print(5 > 3) # True
print(10 == 11) # False

درس ۱۹: آشنایی با مفهوم لیست در پایتون

لیست‌ها مجموعه‌ای از آیتم‌ها هستند که مرتب و قابل تغییرند. آن‌ها با براکت `[]` تعریف می‌شوند.

fruits = ["apple", "banana", "cherry"]
numbers = [1, 2, 3, 4]

مثال ۲: دسترسی به آیتم‌ها از طریق ایندکس

print(fruits[0]) # 'apple'
print(numbers[1:3]) # [2, 3]

درس ۲۰ و ۲۱: عملیات روی لیست‌ها

شما می‌توانید به آیتم‌های لیست دسترسی پیدا کنید، آن‌ها را تغییر دهید، اضافه یا حذف کنید.

my_list = [1, 2, 3]
my_list.append(4) # اضافه کردن آیتم
my_list.remove(2) # حذف کردن آیتم
print(my_list) # [1, 3, 4]

مثال ۲: تغییر آیتم و اضافه کردن لیست دیگر

colors = ["red", "blue", "green"]
colors[1] = "yellow"
print(colors) # ['red', 'yellow', 'green']
colors.extend(["black", "white"])
print(colors) # ['red', 'yellow', 'green', 'black', 'white']

درس ۲۲: متدهای پرکاربرد در لیست

متدهایی مانند `sort()`, `reverse()`, `pop()` و ... برای کار با لیست‌ها مفید هستند.

numbers = [5, 2, 8, 1]
numbers.sort()
print(numbers) # [1, 2, 5, 8]
last_item = numbers.pop()
print(last_item) # 8

مثال ۲: شمردن آیتم‌ها و یافتن ایندکس

fruits = ["apple", "banana", "apple"]
print(fruits.count("apple")) # 2
print(fruits.index("banana")) # 1

درس ۲۳: Tuple در پایتون

تاپل‌ها شبیه به لیست‌ها هستند اما غیرقابل تغییر (immutable) می‌باشند. آن‌ها با پرانتز `()` تعریف می‌شوند.

my_tuple = ("apple", "banana", "cherry")
print(my_tuple[0])
# my_tuple[0] = "orange" # این کار باعث خطا می‌شود

مثال ۲: استفاده از تاپل برای بازگشت مقادیر از یک تابع

def get_coordinates():
    return (10, 20)

x, y = get_coordinates()
print(x) # 10
print(y) # 20

درس ۲۴: مفهوم ست (Set)

ست‌ها مجموعه‌ای از آیتم‌های منحصربه‌فرد هستند که غیرمرتبند. آن‌ها با آکولاد `{}` تعریف می‌شوند.

my_set = {1, 2, 3, 2, 1}
print(my_set) # {1, 2, 3} (تکراری‌ها حذف می‌شوند)

مثال ۲: عملیات ریاضی روی ست‌ها

set1 = {1, 2, 3}
set2 = {3, 4, 5}
union_set = set1.union(set2)
print(union_set) # {1, 2, 3, 4, 5}
intersection_set = set1.intersection(set2)
print(intersection_set) # {3}

درس ۲۵: مفهوم دیکشنری (Dictionary)

دیکشنری‌ها داده‌ها را به صورت جفت‌های کلید-مقدار (key-value) ذخیره می‌کنند.

person = {
    "name": "Mahdi",
    "age": 25,
    "city": "Shiraz"
}
print(person["name"]) # 'Mahdi'

مثال ۲: اضافه و تغییر جفت کلید-مقدار

person["age"] = 26 # تغییر مقدار
person["email"] = "mahdi@example.com" # اضافه کردن کلید جدید
print(person)

درس ۲۶: مفهوم پرینت (Print)

تابع `print()` برای نمایش خروجی در کنسول استفاده می‌شود. با استفاده از f-string می‌توانید متغیرها را به راحتی در رشته‌ها قرار دهید.

name = "Sara"
print("سلام، " + name + "!")
print(f"سلام، {name}!")

مثال ۲: چاپ چندین آیتم

print("نام:", "علی", "سن:", 20)

درس ۲۷: دستور ورودی

تابع `input()` از کاربر ورودی می‌گیرد و همیشه آن را به عنوان یک رشته (string) برمی‌گرداند.

name = input("لطفا نام خود را وارد کنید: ")
print(f"سلام، {name}!")

مثال ۲: تبدیل ورودی به عدد

age_str = input("سن خود را وارد کنید: ")
age_int = int(age_str)
print(f"سن شما در ۵ سال آینده: {age_int + 5}")
